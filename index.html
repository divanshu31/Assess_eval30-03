Understanding Client & Server

The Client sends a request to the server, which processes the request and returns the appropriate respone.

The Journey of a web request

  1. Client enters URL in the browser.
  2. Browser send an HTTP request to the server.
  3. DNS resolves the IP address.
  4. Server processes the request and sends back the response.
  5. Browser renders the web page.

  • HTTP
  • DNS
  • TCP/IP
